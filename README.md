## Web-based Laguna Lake Water Quality Monitoring System


A single-page Next.js web application for visualizing real-time water quality data in Laguna Lake using an interactive map and monitoring station dashboard.

## Overview

This project provides an interactive dashboard that displays water quality metrics and monitoring stations across Laguna Lake. It uses Leaflet maps and OpenStreetMap tiles to visualize environmental conditions such as dissolved oxygen, pH, temperature, and nutrient levels.

The application is implemented as a client-side rendered, one-page dashboard with tab-based navigation.

## Features

- Interactive map with color-coded markers based on water quality
- Summary dashboard of key water quality metrics
- Monitoring stations list with images and descriptions
- Detailed station popups with measured parameters
- Tab navigation (Map / Stations)
- Responsive layout


## Tech Stack

**Client:** React, Next.js, TailwindCSS, Leaflet

**Server:** Vercel


## Installation

Install with npm

```bash
  npm install 
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/Lornz18/witm-llda.git
```

Go to the project directory

```bash
  cd my-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Open in browser
```bash
  http://localhost:3000
```
